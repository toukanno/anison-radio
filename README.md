# anison-radio

Anime song streaming app inspired by Spotify. Play, discover, and enjoy anime music.

## What this repo includes

A lightweight frontend prototype with:

- Discover list of anime songs
- Search by song / artist / anime title
- Simple now-playing state (play/pause/next)

## Run locally

```bash
python -m http.server 8000
```

Then open <http://localhost:8000>.
